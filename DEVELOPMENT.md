# Guida Introduttiva

Questo progetto è un'applicazione frontend costruita con React e Vite. Segui i passaggi seguenti per avviare l'app localmente.

## Prerequisiti

È necessario avere Node.js e npm installati.

## Avvio dell'Applicazione

1. Installa le dipendenze:
   ```bash
   npm install
   ```

2. Avvia il server di sviluppo:
   ```bash
   npm run dev
   ```

Il frontend sarà disponibile all'indirizzo `http://localhost:5173` (o alla porta successiva disponibile).

## Distribuzione (Deployment)

Puoi compilare l'applicazione per la produzione utilizzando:

```bash
npm run build
```

I file compilati si troveranno nella cartella `dist`.
